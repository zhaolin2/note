# git一般提交代码的步骤


```shell
echo "# DataStructures" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/zhaolin2/DataStructures.git
git push -u origin master
```
# 或者提交已经存在的版本库
```shell
git remote add origin https://github.com/zhaolin2/DataStructures.git
git push -u origin master
```
